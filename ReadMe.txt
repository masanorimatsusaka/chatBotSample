
/********************************************/
「自動会話」
/********************************************/
・概要
リクルートが提供している日常会話を可能にするAPI「Talk API」と
Docomoが提供している音声合成API「Powerd by エーアイ」の
２つの外部APIを使用した、2次元のキャラと会話できるソフトになります。

・遊び方の流れ
１、exeファイルを実行してください。
２、右下の入力項目に話しかける言葉を入力してボタンを押してください。
３、左下に返答した文字と画面にいるキャラが口パクしながら返事をしてくれます。

/********************************************/
携わった箇所
/********************************************/
主にプログラム作成、各シーンの作成を行いました。
作成したスクリプトは「自動会話」配下にある「Script」に格納しています。

参考程度にご確認ください。


/********************************************/
現在確認されている不具合
/********************************************/
エディタ上では口パクが問題なく動作しているが、
アプリ上で口パクができません。
OVRLipSyncを使って口パクを行いますが
なにか不具合が出ているか、取得の仕方におかしな点があるかだと思われます。
